Tic Tac Toe

This project involves creating a simple two-player Tic-Tac-Toe game using HTML for structure, CSS for styling, and JavaScript for interactivity and game logic. The game consists of a 3x3 grid where two players can alternately mark cells with "X" and "O"

Features

1. Simple Design:

The game features a clean and minimalist design to ensure focus on functionality and ease of use.

2. Two-Player Mode:

The game allows two players to take turns marking the grid cells.

3. Dynamic Status Display:

A status message updates to show the current player, announce the winner, or declare a draw.

4. Reset Functionality:

A button allows players to reset the game and start a new match.
