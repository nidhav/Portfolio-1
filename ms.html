Restaurant Management System

The Restaurant Management System is a comprehensive application designed to streamline various operations within a restaurant. Developed by Nidhi Yadav, Nishant, Mohit, and Narayan, this system leverages Python's Tkinter library to create a user-friendly graphical interface, and it integrates with Excel to store and manage data efficiently.

Features

1. User-Friendly Interface:

The application provides a clean and intuitive interface using Tkinter, making it easy for staff to navigate and manage restaurant operations.

2. Order Management:

Allows staff to take orders, update them, and track order status.

3. Menu Management:

Facilitates easy updates to the menu items, including adding, removing, and modifying dishes.

4. Billing System:

Generates bills for customers based on their orders, with options for applying discounts and taxes..
