Stone Paper Scissor

This project involves creating a simple single-player Rock-Paper-Scissors game using HTML for structure, CSS for styling, and JavaScript for interactivity and game logic. The game allows a player to play against the computer by selecting rock, paper, or scissors. The computer makes a random choice, and the game determines the winner based on the classic rules of Rock-Paper-Scissors.

Features

1. User-Friendly Interface:

The game features a clean and intuitive design to ensure a pleasant user experience.

2. Single-Player Mode:

The player competes against the computer, which makes random choices

3. Dynamic Result Display:

The result of each round (win, lose, or draw) is displayed dynamically.

4. Score Tracking:

The game tracks and displays the score for both the player and the computer.
